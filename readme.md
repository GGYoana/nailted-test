## Requisitos
* Node js
* Git    
* BDD Postgres (Código en /database/database.sql)

La aplicación está conectada a una base de datos en postgres. Para que funcione es necesario conectar con ella.

## Configuración común
Clone el repositorio e instale las dependencias.
```bash
git clone https://github.com/GGYoana/nailted-test.git
cd nailted-test
npm install 
```

## Pasos para el acceso 
Para iniciar el servidor express, ejecute lo siguiente
```bash
npm run dev
```
Abra http://localhost:3001 y eche un vistazo.
